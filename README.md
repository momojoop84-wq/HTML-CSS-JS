BLOC 6 — Mini projet

Exercice 6
Créer une application simple :
Gestion de tâches (To-Do List)
Fonctionnalités :
Utilisateurs
• ajouter un utilisateur
• modifier utilisateur
• archiver un utilisateur
Taches
• ajouter une tâche
• affecter une tache à un utilisateur
• supprimer une tâche
• marquer comme terminé
• afficher les taches d’un utilisateurs
NB :
- Les taches et les utilisateurs sont garder dans le localStorage
- Le design est laisser a votre appréciation
- L’application doit être responsive
